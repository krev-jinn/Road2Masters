# Road2Masters

## Overview
Road2Masters is an interactive web platform designed to help students follow structured and step-by-step learning roadmaps across multiple technology domains.

The platform provides guided learning paths for:
- Web Development
- Artificial Intelligence
- Cybersecurity
- Data Science
- Mobile App Development

Students can track their progress, access curated video resources, maintain notes, and submit doubts through an integrated Firebase backend.

---

## Features

### User Registration and Validation
- Login system with form validation
- Validation for name, email, and phone number
- User data stored in Firebase Firestore

### Structured Learning Roadmaps
- Four learning phases:
  - Beginner
  - Intermediate
  - Advanced
  - Research
- Organized domain-specific progression paths

### Video Resources
- Topic-wise curated YouTube learning resources
- Popup-based video access system

### Sticky Notes System
- Personal note-taking for every topic
- Multiple sticky note color options
- Notes stored using browser local storage

### Ask a Doubt System
- Students can submit questions related to any topic
- Doubts stored in Firebase Firestore
- Mentor review workflow support

### Milestone Tracker
- Circular progress tracker
- Topic completion checkboxes
- Dynamic progress percentage updates

### Theme Support
- Dark mode and light mode toggle support

### Additional UI Features
- Scrolling announcement banner
- Responsive dashboard interface
- Interactive domain cards and sidebar navigation

## Backend Functionality

The project uses Firebase Firestore as a cloud backend database.

### Stored Data
- User registration details
- Submitted doubts/questions
- Server timestamps for activity tracking

---

## Project Objective

The goal of Road2Masters is to simplify the learning journey for students by providing a centralized and structured roadmap-based learning environment instead of scattered learning resources.
